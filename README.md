## Instalar dependecias

Crear un entorno virtual con python3

```bash
python3 -m venv env
```

Activar el entorno virtual

```bash
source env/bin/activate
```

Instalar las dependencias

```bash
pip install -r requirements.txt
```

## Ejecutar el proyecto

```bash
python3 main.py
```
